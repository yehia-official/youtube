# **App Name**: Yehia Mohammed Tech Hub

## Core Features:

- Home Page Showcase: Display key content sections (Introduction, Latest Videos, Top Courses, Tech Reviews) on the home page.
- Courses Listing: Show all available courses with images, descriptions, and 'Watch Now' buttons, filtering by categories (Frontend, Python, AI, Tools).
- Tech Review Hub: A section where users can find reviews of tools and courses, including star ratings and pros/cons.
- YouTube Integration: Dynamically embed or use the YouTube API to display the latest videos with a subscription call to action.
- Contact Form: A form for users to send direct messages, coupled with social media links for additional contact methods.

## Style Guidelines:

- Primary color: Soft blue (#A7C4D3) for a calming and professional feel, reflecting Yehia Mohammed's educational content.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary color to support a clean and modern look.
- Accent color: Light purple (#B19CD9), a slightly brighter analogous color used for interactive elements like buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean, modern look.
- Use modern, minimalist icons for navigation and key sections.
- A fully responsive design, optimized for various screen sizes.
- Subtle scrolling animations to improve user experience.
